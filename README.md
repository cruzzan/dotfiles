## Dotfiles

I have devided my dotfiles config in a few different modules

* [bash](bash/)
* [git](git/)
* [Vim](vim/)
* GnuPG
    ```bash
    $ ln -s /path/to/dotfiles/gnupg/gpg.conf ~/.gnupg/gpg.conf
    ```
